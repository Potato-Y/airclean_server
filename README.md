## 주요 기능
- 사용자의 기기 조작 요청 처리
- 사용자의 실시간 실내 공기질 요청 처리
- 아두이노와 실시간 통신 처리
  - 아두이노로부터 실시간 실내 공기질 정보를 받음
  - 사용자의 기기 조작 명령어 전달
